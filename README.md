## 📖 项目初衷 (Introduction)

秉持互联网开源精神，也致力于为那些学术能力强但不擅长信息收集的同学提供更公平的备考环境。

过去三年，我发现很多优质的复习资料（历年卷、笔记、重点整理）往往只在小圈子内流传。为了打破这种“信息差”，**我将近年收集并筛选出的、与考试相关度最高且最有帮助的资料整理并上传至本仓库**，供大家复习参考，帮助大家把更多精力投入到更有意义的事情中。

这里没有真正的“秘密”，只有前人留下的智慧火种。🔥
